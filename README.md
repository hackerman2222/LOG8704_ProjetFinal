# Projet Final - LOG8704

## Description
Il s’agit d’un projet XR développé dans le cadre du cours LOG8704.

L’objectif de ce README est de fournir toutes les informations nécessaires pour **builder et tester l’application**, ainsi que des détails sur les appareils et plateformes utilisées.

---

## Contenu du dépôt
- **/Assets** : Contient tous les assets nécessaires (textures, modèles 3D, sons…).  
- **/Scripts** : Contient les scripts C# utilisés dans le projet.  
- **/Scenes** : Contient la scène principale à ouvrir pour tester le projet.  

---

## Scène à utiliser
Pour tester le projet, ouvrez la scène suivante dans Unity :  /Scenes/Project_Scene.unity

---

## Configuration requise
- **Unity version** : Nous avons utilisé la version 6000.0.54f1 pour le développement. 
- **Plateformes testées** :  
  - Casque Meta Quest 3.

---

## Instructions pour lancer le projet
```bash
1. Clonez le dépôt :  
git clone https://github.com/hackerman2222/LOG8704_ProjetFinal

2. Ouvrez Unity Hub et ajoutez le projet.

3. Ouvrez la scène principale : Assets/Scenes/Project_Scene.unity

4. Connecter votre casque.

5. Appuyez sur File--> Build Profiles --> choisir la plateforme android --> Run device (Occulus Quest 3 connecté) --> Build And Run.

6. Suivez les instructions affichées dans l'interface pour l'utilisation de l'application.
